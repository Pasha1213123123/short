# ➕ Перенос кнопки добавления видео в настройки

Задача по перемещению функционала загрузки видео (Floating Action Button) из основного экрана ленты в экран настроек.

---

## Задача

**MOVE-001**: Перенос кнопки добавления видео в экран настроек

- **Описание:** Убрать плавающую кнопку (FAB) с главного экрана `ShortsPageView`. Добавить возможность инициировать загрузку видео из экрана `SettingsScreen`.
- **Цель:** Очистить интерфейс просмотра видео от лишних элементов, сосредоточив управление приложением в одном месте.
- **Приоритет:** Улучшение (UI/UX).

### Требования

- Удалить `FloatingActionButton` из `lib/screens/shorts_page_view.dart`.
- Добавить новый пункт меню (ListTile) в `lib/screens/settings_screen.dart` с надписью «Загрузить видео» (Upload Video).
- Сохранить логику управления плеером: при переходе на экран загрузки текущее видео должно ставиться на паузу, а при возврате — возобновляться.
- Обеспечить корректную навигацию и использование локализации.

---

## Способы реализации (подробно)

### 1. Изменение экрана ленты (ShortsPageView)
**Файл:** `lib/screens/shorts_page_view.dart`
- Найти свойство `floatingActionButton` в виджете `Scaffold`.
- Полностью удалить определение `FloatingActionButton` и `floatingActionButtonLocation`.
- Это освободит правый нижний угол экрана, сделав просмотр более погруженным.

### 2. Изменение экрана настроек (SettingsScreen)
**Файл:** `lib/screens/settings_screen.dart`
- Внутри `ListView` добавить новый элемент `ListTile` перед разделом версий или после настроек темы.
- **Параметры ListTile:**
    *   `leading`: Иконка `Icons.add_circle_outline` или `Icons.video_call`.
    *   `title`: Текст из локализации `loc.uploadVideo`.
    *   `trailing`: Иконка стрелки `Icons.chevron_right`.
    *   `onTap`: Реализация логики перехода.

### 3. Логика управления видео (Critical Path)
Поскольку управление плеером при навигации важно для стабильности, необходимо аккуратно перенести код из старой FAB в новый `onTap`:
1.  Получить текущий контроллер через `ref.read(currentVideoControllerProvider)`.
2.  Если видео играет — поставить на паузу.
3.  Выполнить `Navigator.push` на `UploadScreen`.
4.  После возврата (await) добавить задержку `AppConstants.navigationDelay`.
5.  Снова получить контроллер (он мог измениться) и запустить воспроизведение, если видео на паузе.

### 4. Альтернативный вариант UI в настройках
Вместо обычного списка можно использовать:
- **Button:** Большая кнопка `ElevatedButton.icon` в верхней или нижней части экрана настроек.
- **Grid Item:** Если экран настроек будет переделан в сетку, добавить отдельную плитку «Создать».
