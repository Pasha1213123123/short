# Детальный план реализации FAB_LIKE_SHARE_FAVORITE_TASK

Этот план описывает замену трех отдельных кнопок (Лайк, Поделиться, Избранное) одной компактной плавающей кнопкой с раскрывающимся меню (Speed Dial).

## 1. Исследование и выбор решения
- [ ] Оценить текущую структуру `_buildRightSideBar` в `lib/screens/short_player_screen.dart`.
- [ ] Выбор реализации: 
    *   **Вариант А:** Использование пакета `flutter_speed_dial` (наиболее стабильный и функциональный).
    *   **Вариант Б:** Кастомная анимация на базе `Stack` и `AnimatedContainer` (максимальный контроль, но больше кода).
    *   *Рекомендация:* Использовать `flutter_speed_dial` для обеспечения плавных анимаций и поддержки жестов "из коробки".

## 2. Подготовка зависимостей
- [ ] **Файл:** `pubspec.yaml`
- [ ] Добавить `flutter_speed_dial: ^7.0.0` (или актуальную версию).
- [ ] Выполнить `flutter pub get`.

## 3. Рефакторинг правой панели плеера
- [ ] **Файл:** `lib/screens/short_player_screen.dart`
- [ ] Удалить методы `_buildLikeButton`, `_buildBookmarkButton`, `_buildShareButton`.
- [ ] Создать метод `_buildUnifiedFab()`, который будет возвращать `SpeedDial`.
- [ ] В `SpeedDial` добавить три `SpeedDialChild`:
    1.  **Like:** Использовать логику из старой кнопки, отображать иконку `favorite`/`favorite_border` в зависимости от состояния `likedVideosProvider`.
    2.  **Favorite:** Отображать иконку `bookmark`/`bookmark_border` в зависимости от `bookmarkedVideosProvider`.
    3.  **Share:** Копировать ссылку в буфер обмена.

## 4. Сохранение визуальной обратной связи
- [ ] Обеспечить обновление иконок в меню Speed Dial при нажатии на них (реактивность через Riverpod).
- [ ] Сохранить Haptic Feedback (виброотклик) при каждом действии.
- [ ] Добавить текстовые метки (Labels) к кнопкам внутри меню для улучшения UX.

## 5. Валидация
- [ ] Проверить, что кнопки в меню корректно меняют цвет при лайке/сохранении.
- [ ] Проверить, что меню закрывается после нажатия (или остается открытым, если это удобнее — настроить `closeManually`).
- [ ] Убедиться, что функционал Firebase Analytics не пострадал.
