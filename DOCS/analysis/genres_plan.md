# Детальный план реализации GENRES_FROM_MUX_API_TASK

Этот план описывает переход от захардкоженного списка жанров к динамическим категориям, извлекаемым напрямую из метаданных видео (Mux API).

## 1. Обновление сервиса API
- [ ] **Файл:** `lib/services/mux_api_service.dart`
- [ ] В методе `createDirectUploadUrl` расширить параметр `passthrough`. Теперь он должен принимать объект, содержащий не только `title` и `description`, но и `genres` (List<String>).
- [ ] Убедиться, что при создании загрузки жанры корректно сериализуются в JSON для поля `new_asset_settings.passthrough`.

## 2. Парсинг динамических метаданных
