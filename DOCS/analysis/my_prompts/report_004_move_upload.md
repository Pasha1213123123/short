# Отчет по изменениям #4: Перенос загрузки видео в настройки

**Дата:** 9 марта 2026 г.

### Задача:
Перенос функционала добавления видео из главного экрана в раздел настроек (MOVE-001).

### Выполненные действия:
1.  **Очистка главного экрана:**
    *   В `lib/screens/shorts_page_view.dart` удален виджет `FloatingActionButton` и его позиционирование. Это позволило сделать интерфейс просмотра более чистым и погруженным.
2.  **Обновление экрана настроек:**
    *   В `lib/screens/settings_screen.dart` добавлен пункт меню `ListTile` с иконкой `video_call`.
    *   Реализована логика навигации: при нажатии на «Загрузить видео» текущий плеер ставится на паузу. После возвращения из экрана загрузки воспроизведение возобновляется с небольшой задержкой для плавности.
3.  **Локализация:**
    *   Использована существующая строка `loc.uploadVideo` для обеспечения поддержки нескольких языков.

### Измененные файлы:
*   `lib/screens/shorts_page_view.dart`
*   `lib/screens/settings_screen.dart`

### Результат:
Интерфейс ленты стал более современным, а функция создания контента теперь логично расположена в разделе управления (настройках).
